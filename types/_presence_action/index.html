blang[ruby].
Ably::Models::PresenceMessage::ACTION is an enum-like value representing all the Realtime Presence states & events. ACTION can be represented interchangeably as either symbols or constants.

h4. Symbol states ```[ruby] :absent # => 0 (reserved for internal use) :present # => 1 :enter # => 2 :leave # => 3 :update # => 4 ``` h4. Constant states ```[ruby] PresenceMessage::ACTION.Absent # => 0 (internal use) PresenceMessage::ACTION.Present # => 1 PresenceMessage::ACTION.Enter # => 2 PresenceMessage::ACTION.Leave # => 3 PresenceMessage::ACTION.Update # => 4 ``` h4. Example usage ```[ruby]
  1. Example with symbols
    presence.on(:attached) { … }
  1. Example with constants
    presence.on(Ably::Models::PresenceMessage::ACTION.Enter) { … }
  1. Interchangeable
    Ably::Models::PresenceMessage::ACTION.Enter == :enter # => true
    ```
